# what is html5 ?

  1. html5 is an advanced version of html
  2. html5 is used to start with <!DOCTYPE html>
  3. html5 used some new html elements or tags 
  4. html5 support latest tags 
  5. html5 used for media type of tags 
  6. hml5 file extension .html or .htm


# structures of html5 or boiler plate of html5

 ```
<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Document</title>
</head>
<body>

</body>
</html>

 ```
