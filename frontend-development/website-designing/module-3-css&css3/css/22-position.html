CSS Position Property

What is Position in CSS?

The position property in CSS specifies how an element is positioned in the document. It determines how an element's location is calculated on the page and whether it participates in the normal document flow or is taken out of the flow.

Types of CSS Position Values

1. Static (Default)

Elements are positioned according to the normal document flow. Properties like top, bottom, left, and right have no effect.

This is a static positioned element (default behavior)

2. Relative

An element is positioned relative to its normal position. The element still occupies space in the normal flow, but can be offset using top, bottom, left, and right properties.

This element is relatively positioned (moved 20px down and 30px right)

3. Absolute

An element is removed from the normal document flow and positioned relative to its nearest positioned ancestor. If no positioned ancestor exists, it's positioned relative to the document body. Uses top, bottom, left, and right for placement.

This is a relatively positioned container
Absolutely positioned element (top: 10px, right: 10px)

4. Fixed

An element is removed from the normal flow and positioned relative to the viewport. It stays in the same position even when the page is scrolled. Uses top, bottom, left, and right for placement.

Note: A fixed element is visible in the top-right corner of the screen. Try scrolling to see it stay in place!

5. Sticky

An element is positioned based on the user's scroll position. It toggles between relative and fixed positioning depending on where the element is in the viewport. Works with top, bottom, left, and right.

This is a sticky positioned element (stays at top when scrolling)
