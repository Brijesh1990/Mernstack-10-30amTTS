# what is difference between css and css3 ?

CSS (Cascading Style Sheets) is the standard style sheet language used to describe the presentation of HTML documents. CSS3 is the latest major version of CSS and adds new features and modules such as:

- Selectors level 3 and 4
- Rounded corners with `border-radius`
- Box shadows with `box-shadow`
- Gradients
- Transitions and animations
- Media queries for responsive design
- Flexbox and Grid layout modules
- image filter effects 
- paragraph count 


In short, CSS is the general styling language, while CSS3 refers to the newer additions and improvements in the latest CSS specification.
